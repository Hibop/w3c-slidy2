同步工具 qrsync

建立配置文件（比如命名为 config.json）

{
    "access_key": "<公钥>",
    "secret_key": "<私钥>",
    "bucket": "<空间名称>",
    "sync_dir": "<本地同步目录（绝对路径）>",
    "domains": ["foo.dn.qbox.me", "bar.qiniudn.com"],
    "debug_level": 0
}

执行同步上传操作

./qrsync /path/to/config.json // for Linux or MacOS

qrsync.exe /path/to/config.json // for Windows

文件访问方式

http://<绑定域名>/<文件名(可带斜杠)>

http://foo.dn.qbox.me/a.txt
http://foo.dn.qbox.me/a/b/c.txt

详见：http://docs.qiniutek.com/v3/tools/qrsync/

开发辅助调试工具 qboxrsctl

qboxrsctl login <email> <password>
qboxrsctl // 查看命令列表

详见：http://docs.qiniutek.com/v3/tools/qboxrsctl/
